# Alias Setup Documentation

## Alias Definition
```bash
alias ch='sudo chmod -R 777 $1 && cd $1'
```

## GitHub Repository

## Setup Command
Run this command on any new system to set up the alias:
```bash
curl -sSL https://raw.githubusercontent.com/suppg022312/test/main/setup_aliases.sh | bash
```
